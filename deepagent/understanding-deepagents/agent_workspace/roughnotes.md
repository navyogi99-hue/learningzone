# Rough Notes: Beginner's Guide to Algebra

## 1. Learner's Confusion
- Why do letters suddenly appear in math?
- Students are comfortable with numbers (e.g., 5 + 3 = 8), but when 'x' and 'y' are introduced, it looks like a foreign language.
- This creates anxiety and confusion. Let's address this fear directly and validate it.

## 2. Simplifying the Problem
- What is algebra in simple terms? It is just finding the missing number.
- In early grade math, we had questions like: [ ] + 3 = 8.
- Algebra is the exact same thing, but instead of a blank box [ ], we use a letter like 'x'.
- E.g., x + 3 = 8.
- Explain terms like "variable" (the letters holding a spot) and "equation" (the balance scale).

## 3. Practical Example
- Let's use a grocery store scenario.
- Buying 3 chocolate bars and a bottle of juice.
- Juice costs $2. Total spent is $14.
- Let 'c' represent the cost of one chocolate bar.
- Equation: 3c + 2 = 14.
- Solve step-by-step:
  - Subtract 2 from both sides: 3c = 12.
  - Divide both sides by 3: c = 4.
  - So, each chocolate bar costs $4.
- This makes algebra feel concrete and useful.

## 4. Common Mistakes
- Not keeping the equation balanced (doing something to one side but not the other).
- Confusion with operations: when to add vs. subtract, multiply vs. divide (inverse operations).
- The "Squish" rule: 3x means 3 multiplied by x, not 3 + x or the digit 3 next to x.

## 5. Short Summary
- Keep the balance.
- Use opposite operations.
- Letters are just placeholders for secrets we want to discover.
